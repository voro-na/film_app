# Film App
Front-end приложение для поиска фильмов. Данные берутся из открытого API: https://kinopoiskapiunofficial.tech/documentation/api/
Реализована возможность регистрации пользователей с помощью Firebase
***
## Стек технологий:
- Typescript
- SCSS
- React, React Hooks
- React-router-dom
- Mobx
- Axios
